# Project Memory

## Core
Carousel-focused SaaS for Brazilian medical professionals. PT-BR only.
Instrument Serif headings, Inter body. Off-white bg, graphite text, sage green accent (#3B4A3A / HSL 130 12% 26%).
Lovable Cloud enabled. AI via edge functions (agent-carrossel, suggest-carousel-topics).
3 screens only: Perfil (/setup), Carrossel (/carrossel), Biblioteca (/biblioteca).
