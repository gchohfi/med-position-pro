## Typography
- Headings: Instrument Serif (Google Fonts) — elegant, editorial serif
- Body: Inter — clean sans-serif with OpenType features (cv11, ss01)
- Display sizes: text-display (3.5rem) and text-display-sm (2.5rem)

## Color Palette (HSL)
Light mode:
- Background: 0 0% 100% (pure white)
- Foreground: 0 0% 7% (near black)
- Accent: 220 72% 53% (refined blue)
- Muted: 220 6% 96%
- Border: 220 6% 92%
- Secondary: 220 6% 96%

Dark mode:
- Background: 0 0% 4%
- Foreground: 0 0% 93%
- Accent: 220 72% 58%

## Design Direction
- Minimalist premium — inspired by Apple, Linear, Stripe
- Generous whitespace, clean typography
- Subtle borders (border/40 opacity)
- No heavy shadows, no gradients
- Radius: 0.625rem (--radius)
- Antialiased text rendering
